Enterprise Direction

开源版当前适合承诺的能力

这个页面强调的是“基础支持”而不是复杂文档的高保真恢复，因此对外展示也保持克制和清晰。

基础格式支持支持 PDF、图片、`DOCX`、`XLSX`、`PPTX` 的基础解析与验证。
双输出统一输出 `Markdown` 与 `JSON`，适合人工检查和系统接入。
可观察 pipeline可直接查看 diagnostics，定位是否走 OCR、是否发生回退、各阶段节点变化。
可嵌入集成除 Demo 外，还提供 CLI 和 Java SDK，适合本地验证与二次开发。

如果你需要的不只是基础支持，而是复杂场景下的效果和交付保障

`SmartDoc-Flow` 的开源版适合作为文档解析前置层与集成入口。对于复杂财报、法务文档、科研论文、低质量扫描件和企业部署交付，企业增强版会更适合。

  • 更强 OCR Provider 路由、质量评分、失败回退与扫描增强
  • 复杂表格恢复、多栏阅读顺序、高价值结构语义恢复
  • 私有化部署、专项规则包、长期效果优化与支持服务
